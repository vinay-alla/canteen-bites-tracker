# 🍔 AU I Block Canteen App

A modern, feature-rich **canteen ordering web application** built for the AU I Block campus canteen. Browse the menu, add items to cart, apply discount coupons, and earn reward points — all in one place.

## ✨ Features

- 🥗 **Menu Browsing** — Categorized into Vegetarian & Non-Vegetarian sections
- 🔍 **Search** — Instantly search for snacks and meals
- 🛒 **Cart Management** — Add, remove, and manage your order
- 🎟️ **Coupons & Discounts** — Apply coupon codes (10% OFF, 15% OFF, 18% OFF deals)
- 🏆 **Rewards System** — Earn coins with every order (Rewards Tiers included)
- 📦 **Stock Status** — Real-time item availability (Available / Running Low / Out of Stock)
- 🔄 **Live Updates** — Menu updates with timestamp
